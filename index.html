如何卸载BetterIntellij？

1、在菜单栏打开 File -> Setting

2、打开Plugins，在已安装列表（Installed）中找到 BetterIntellij，点击鼠标右键，会弹出选择菜单，点击 Uninstall，便可卸载。

3、完成后，点击重启IDEA，此时，BetterIntellij便卸载完成。
